DropDowns
Dropdown is a menu which gives us an opportunity to select one or more options

Types of dropdowns
1) Single select: You can select only one option
2)Multiple select: You can select multiple options

Handling dropdowns:
1.-click on dropdown and click on option
2.- Using Select Class in order to use select class we need to make sure our dropdown has a select tag.

Selecting an option from a dropdown can be done in 3 ways.

1) by Index
2)by visible Text
3)by value

Whenever you interact with multiple elements you store them inside the container such as List, Set...
